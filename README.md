## Principali SW
* [Gimp](https://www.gimp.org/): programma utilizzato per creare le carte
* [Libre Office Write](https://it.libreoffice.org/): utilizzato per impaginare per la stampa (soluzione un po' brutale ma funzionante!)
* [Valentina](https://smart-pattern.com.ua/en/valentina/download/): utilizzato per creare la scatola che ha delle misure variabili che possono facilmente essere modificate per ottenere la versione eventualmente + grande o + piccola (sarebbe un software per fare carta modelli per vestiti ma funziona benissimo anche per packaging)

## Installare font su linux
Copiarli nella cartella /usr/local/share/fonts
